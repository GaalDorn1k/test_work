Тестовое задание для РИТ

Скрипт может быть запущен командой "python test2.py"
Для установки приложения запустите скрипт "setup.py": "python setup.py develop"
После установки приложение запускается командой "test2"

ВНИМАНИЕ!!!
Приложение работает только с файлами, имеющими имена "Приложение A", "Приложение B", "Приложение C"
После запуска приложения будет предложено будет выбрать папку, в которой лежат эти файлы
PNG-отчет с оценкой трудозатрат будет записан в ту же папку
