Тестовое задание для РИТ


Для запуска приложения активируйте скрипт "test.py": пропишите в консоли "python test.py"

ВНИМАНИЕ!!!
Приложение работает только с файлами, имеющими имена "A", "B", "C"
После запуска приложения будет предложено будет выбрать папку, в которой лежат эти файлы
PNG-отчет с оценкой трудозатрат будет записан в ту же папку



Количесво часов, потраченных на выполнение задания - 6:
3 часа		- на выполнение заданий в JupiterNotebook
2 часа		- на написание и отладку кода
1 час		- на коментирование, проверку и обработку исключений
